# 학습 로드맵

## 1단계: 계산과 실행 모델

- 자료구조와 알고리즘: 복잡도, 연결 리스트, 스택, 해시, 트리, Trie, 그래프, 정렬, 탐색
- 컴퓨터 구조: 명령어 파이프라이닝, 파이프라인 위험, 캐시, 참조 지역성
- Operating System: 프로세스, 스레드, 동기화, 가상 메모리, I/O
- Java: JVM, GC 알고리즘, 메모리 모델, 동시성, Stream, Optional, Reflection, Record, 객체 비교

## 2단계: 요청과 데이터

- 네트워크: TCP/IP, DNS, TLS, 로드밸런싱
- Web/HTTP: HTTP 메시지, REST, 쿠키/세션, CORS, HTTP/2, 프록시, 멱등성
- Database: 트랜잭션, 격리 수준, MVCC, 인덱스, 실행 계획, 복제, NoSQL

## 3단계: 백엔드 프레임워크와 저장소

- 객체지향: 캡슐화, 추상화, 다형성, SOLID, 디자인 패턴, 일급 컬렉션
- Spring: IoC/DI, Bean 관리, AOP, MVC, 트랜잭션 전파, JPA, Boot 자동 설정
- Redis: 자료구조, 캐시 패턴, 분산 락, RDB/AOF, Sentinel, Cluster, Stream
- Kafka: 토픽, 파티션, 컨슈머 그룹, offset, rebalancing, 전달 보장, Streams/Connect

## 4단계: 운영 가능한 시스템

- Security: 인증, 인가, JWT, SQL Injection, CSRF, 암호화, HTTPS, 해시, OWASP
- 분산시스템: CAP, 일관성, 복제, 샤딩, 합의, 멱등성, Event Sourcing, CQRS, Outbox, Bulkhead, Circuit Breaker
- DevOps/Cloud: Linux, 컨테이너, Gradle, 테스트 전략, IaC, CI/CD, 무중단 배포, Health Check, Micrometer, Serverless, 관측성, 확장성과 장애 대응
